# Planning Artifacts

This directory is intentionally sparse. Active task plans belong in the session state, not in the repository. Add files here only when a plan becomes durable reusable knowledge for future agents, such as a standing planning template, a recurring review checklist, or a long-lived architecture decomposition that is not owned by `docs/` or `packages/docs/`.

Use `src/AGENTS.md` as the runtime owner map. Guppi plans should state raw-input ownership, markdown source-of-truth rules, session behavior, autonomy boundaries, validation evidence, and migration or rollback boundaries without copying that file inventory.
